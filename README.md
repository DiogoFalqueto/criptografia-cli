<h1>Criptografia CLI</h1>
<p>Pacote para criptografar arquivos por linha de comando(CLI), feita em NodeJS.</p>
<h2>Instalação</h2>
<code>npm i -g @diogofalqueto/criptografia-cli</code>
<h2>Utilização</h2>
<p>Criptografar arquivo:</p>
<code>encript -e -i &it arquivo a ser criptografado &gt -o &it nome do arquivo de saida &gt</code>
<p>Descriptografar arquivo:</p>
<code>encript -d -i &it arquivo a ser descriptografado &gt -o &it nome do arquivo de saida &gt</code>
